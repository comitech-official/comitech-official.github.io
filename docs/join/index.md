## 「こみてっく！」開催決定！！
　こみてっく！は、2026年秋の[工大祭](https://koudaisai.jp/)にて、以下の内容で同人即売会を開催します！！

### とき・ところ
- 2026年 10月10日(土)・11日(日) 10:00 ~ 16:00
- 西9号館3階 W9-327教室 ([東京科学大学 大岡山キャンパス](https://www.titech.ac.jp/public-relations/about/campus-maps/ookayama))

### 参加方式
- **出展参加**：机に自分たちのブースを作って頒布を行います。いわゆる「サークル参加」です。
    - 本ページで参加方法を詳しく説明します。**最後までお読みください**。
- **委託参加**：当日参加が難しい方に代わって、運営準備会が代理で頒布を行う参加方式です。
    - \>\> [こちらのページをご確認ください。](/join/itaku)
- **合同誌への寄稿**：こみてっく！準備会が発行するオールジャンル合同誌に寄稿する方式です。
    - 出展参加・委託参加予定の方もご参加いただけます。
    - \>\> [こちらのページをご確認ください。](/join/kikou)
- **当日来場**：出展はせず、当日の会場で好みの頒布物を探し購入したり、制作者との交流を楽しむ参加方法です。

### 出展参加者 当日までの流れ
- 8/8(土) 23:59　参加登録・サークルカット提出締切
- 8/15(土) 抽選結果・スペース番号を通知
- 10/9(金) 前日搬入・設営
    - 前日搬入・設営の参加は任意です。詳細は追ってご案内します。
- 10/10(土)・11(日) 9:00 出展参加者入場開始
- 10/10(土)・11(日) 10:00 - 16:00 開催

　下記の**募集要項を必ずお読みのうえ**、こちらの参加登録フォームにご記入ください。

## 出展参加 募集要項
### 参加資格
　申込責任者が以下の条件をすべて満たしている必要があります。
1. 高校生を除く18歳以上
1. 「こみてっく！」公式サイトを閲覧し、当会が用意するGoogleフォームに記入できる
1. 当会とメールまたは電話（緊急時のみ）による連絡が可能
1. 当会の連絡に対し、日本語または英語での対応ができる

### 開催日程
　上記の通り。

### 出展参加者 参加費
　参加費は下表の通りです。参加当日に直接支払いをお願いいたします。現金でのみ承ります。

| | 1日参加 | 2日参加 |
| ---- | ---- | ---- |
| 1スぺース | 1,500円 | 2,000円 |
| 2スペース | 2,500円 | 3,000円 |

### 頒布・展示可能なもの
- 出展者または出展団体による、自主制作物。
    - ジャンル・形態を問いません。
    - 自作の商業作品の頒布についてはご相談ください。

### 頒布・展示に注意が必要なもの
- 画像生成AIを用いた作品
    - こみてっく！では、独自基準によるルールを設けています。
    - \>\> [こちらのページをご確認ください。](/join/ai)

### 頒布・展示できないもの
- 自主制作物でないもの(古書、古物等)
- 18歳未満の者による閲覧、購入、入場が法律や東京都青少年の健全な育成に関する条例で禁止されている内容
- 食品
- 医薬品類(化粧品含む)・毒劇物
- すぐに発火できる状態の器具
- 大きな音や強い匂いの出るもの・他ブースや会場を汚損するもの
- 誹謗中傷を含む物品
- その他[工大祭2026参加団体規約・工大祭2026参加団体募集要項](https://join.koudaisai.jp/documents/)で販売が規制されているもの
- その他法律で販売、配布、頒布が規制されているもの
    - ただし、規制に準じた正当な手続きが取られている場合を除きます

### 抽選について
- 申込数によっては、一部サークルにブースを用意できない場合がございます。
- 2スペースで申込をされても、都合により1スペースでの登録となる場合がございます。
　抽選に際して、代表者が東京科学大学に所属している出展者の優遇措置を取らせていただきます。

　抽選から漏れた場合、[委託参加](/join/itaku)の希望を尋ねさせていただきます。

### その他注意
- 工大祭実行委員会からの求めにより、日本円、および円建て以外の決済はご遠慮いただいております。使用予定の決済手段の事前申請をお願いしております。また、使用する決済手段の各種規約に反しない必要があります。
    - 例： PayPayの個人間送金機能を営業活動に転用すること
- 寄付・カンパ行為の有無
    - 過度なカンパの呼びかけ等、他の参加者の迷惑になる行為は禁止しております。
- 本イベント会場での来場者の個人情報収集を禁止します。整理券等を使用される際は、個人情報の収集を伴わない手段をお願いしております。
- 本ページの記載内容のほか、工大祭の[各種規約](https://join.koudaisai.jp/documents/)の遵守をお願いしています。併せてご確認ください。
- 本イベントの運営に関し、工大祭実行委員会の協議等により本ページの内容が変更となる場合がございます。その場合はメール等で連絡いたしますので、恐れ入りますがご対応をお願いします。
- こみてっく！準備会と工大祭実行委員会、及びそれらの構成員は、本イベントに起因し、又は関連して損害が発生した場合であっても、その損害について責任を負わないものとします。ただし、その損害の全部又は一部がこみてっく！準備会と工大祭実行委員会、及びそれらの構成員の重大な過失により生じた損害については、この限りではありません。

## 申込に必要な情報
　各項目の詳細は申込フォーム(link)をご確認ください。
### 0. 申込者の情報
　工大祭内での開催に必要な内容は、工大祭実行委員会に共有することがあります。
- 代表者の氏名や連絡先
- サークル名
    - サークル名とは：団体の場合は団体名を入力してください。同人即売会文化では、個人で参加をする場合でも、屋号に相当する「サークル名」を用いることが一般的です。初めて参加される方も、ぜひこの機会に個性的なサークル名を考えてみてください！
- 公開される名前
    - ハンドルネーム・ペンネーム等

### 1. サークルカット
　出展者をアピールするためのポスターに相当します。以下の項目を含めてください。
- サークル名
- 頒布物の内容
- 頒布物を紹介するイラスト、文章等
\>\> [詳細はこちら](/join/sakuka)

### 2. 頒布予定物の内容
- タイトル
    - 題名、アルバム名、作品名など
    - 頒布価格
- 価格変更の有無と、その方法
- 頒布物の概要
- 画像生成AIの使用予定があるか
    - 使用予定の際には追ってご連絡します。使用の程度等、当団体が使用形態を把握させていただきます。詳細は[こちら](/join/ai)をご覧ください。

### 3. 電源使用の有無
　会場の電源容量から、ご期待に沿えないケースがございます。ご了承ください。

### 4. その他項目
- 落選時に、委託参加を希望するか否か
- サークル自薦文章(100字以内、任意)
    - 「こみてっく！」公式Twitter(現X)等にて内容を紹介させていただきます。
- ボランティア参加の有無
    - 当会の円滑な運営のため、当日のボランティアスタッフを募集しております。
    - 内容は1時間程度の参加者誘導等の業務を予定しています。
- その他、上記注意事項に応じた確認内容
